# Frontend - Finanças Cristãs

Vue 3 + TypeScript + TailwindCSS + DaisyUI

## Instalação

```bash
npm install
```

## Configuração

```bash
cp .env.example .env
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```

Frontend: http://localhost:5173

## Validação Local (Dia 5)

```bash
npm install
npm run lint
npm run test
```
